## Condições físicas: 
São condições provenientes do meio físico contra o próprio físico normalmente causadas por armas e ataques dos mais variados, ou também de péssimas escolhas, podendo ser condições de paralisia, lesão, ou doença

  

- **Agarrado**
  O alvo fica imobilizado e não pode fazer ataques com armas grandes ou fazer gestos para magia além de não poder se mover para longe de quem o agarrou
  — — — — — — // — — — — — — — — — // — — — — — — — — — // — — — — — —
- **Enjoado**
  Você não pode se concentrar em magias, e recebe -2 em testes que precisem de coordenação
  — — — — — — // — — — — — — — — — // — — — — — — — — — // — — — — — —
- **Lento**
  Você perde metade do seu deslocamento, e metade do valor de celeridade da sua Defesa
  — — — — — — // — — — — — — — — — // — — — — — — — — — // — — — — — —
- **Paralisado**
  Você só pode realizar ações puramente mentais (pergunte ao mestre) e perde toda sua celeridade da defesa
  — — — — — — // — — — — — — — — — // — — — — — — — — — // — — — — — —
- **Atordoado**
  Cambaleando você se torna incapaz de realizar ataques ou se concentrar em magia, mas continua executando outras ações
  — — — — — — // — — — — — — — — — // — — — — — — — — — // — — — — — —
- **Inconsciente**
  Você apagou, pode fazer um teste de vontade (DT varia de acordo com a habilidade) caso não passe continua no chão sem poder fazer nenhuma ação
  — — — — — — // — — — — — — — — — // — — — — — — — — — // — — — — — —
- **Por um fio**
  Assim como inconsciente mas causada por um ferimento fatal, pode levar a perda do personagem, você toma 1d6 de dano por rodada, caso passe em uma DT 15 ou igual ao valor negativo de NV (o que for maior) com um teste de fortitude para de tomar o dano e fica inconsciente até alguém o estabilizar com magia ou Medicina
  — — — — — — // — — — — — — — — — // — — — — — — — — — // — — — — — —
- **Sangramento**
  Toma 1d4 de dano todo turno magia de cura ou teste de medicina para a condição
  — — — — — — // — — — — — — — — — // — — — — — — — — — // — — — — — —
- **Hemorragia**
  Toma 1d10 de dano todo turno, só pode ser parado por magia ou remédios e poções específicos
  — — — — — — // — — — — — — — — — // — — — — — — — — — // — — — — — —
- **Laceração Grave**
  Um ferimento preocupante, perde 1d4 de vida permanente por turno precisa ser parado por magia ou pelas habilidades corretas o quanto antes. a vida perdida pode vir a ser recuperada por habilidades específicas, mas até lá estará com uma penalidade na vida máxima (os casos dessa condição são raros mas não nulos) 
  — — — — — — // — — — — — — — — — // — — — — — — — — — // — — — — — —
- **Fratura*** (Animais sem ossos não sofrem fratura)
  Uma fratura é um dano direto a um membro específico, pode ser atingido pela ação “focar” ou outras habilidades semelhantes

  Fratura nos braços irá causar penalidade de -2 nos ataques, 

  Caso os dois braços sejam atingidos a penalidade se torna -5 e cada ataque corpo a corpo que fizer irá lhe causar 1d4 de dano como rebote.

  *No caso de criaturas com mais braços a penalidade soma de -1 em -1 até que atinja o  -5 que é a fratura de todos os membros

  Fratura nas pernas irá causar uma penalidade de 3m de deslocamento e -2 ao fazer testes de reflexo, caso as duas pernas estejam quebradas, adquire a condição de caído e fica com apenas metade do deslocamento além de receber -5 no deslocamento

  Uma fratura nas costelas irá causar penalidade -3 em testes baseados em corpo

  Podem existir outros casos onde fratura se aplica com penalidades diferentes, a variar do teor da mesa
   — — — — — // — — — — — — — — — // — — — — — — — — — // — — — — — —
- **Desmembrado**
  Basicamente o mesmo de fraturado, porém, para parar a condição precisa receber uma magia de cura de 2º círculo ou superior em no máximo 1 dia, ou pode adquirir uma prótese, além do padrão, perde ⅛ do NV máximo
  — — — — — // — — — — — — — — — // — — — — — — — — — // — — — — — —
- **Destruição de membro**
  Uma situação ainda pior do descrito anteriormente, não tem cura e as feridas removem ⅙ do seu NV máximo, adquirir próteses se torna mais custoso pelo estado final do ferimento
  — — — — — // — — — — — — — — — // — — — — — — — — — // — — — — — —
- **Cansado**
  Você está cansado demais para exercer suas atividades habituais, resultando em uma penalidade de -3 em todos os testes que fizer.
  — — — — — // — — — — — — — — — // — — — — — — — — — // — — — — — —
- **Exausto**
  Você está cansado demais para executar qualquer tarefa, recebendo desvantagem em todos os testes que realizar
  — — — — — // — — — — — — — — — // — — — — — — — — — // — — — — — —
- **Cego¹**
  Um personagem cego perde parte das suas capacidades de percepção, recebendo -5 em duelo e percepção e se tornando incapaz de ter certeza de um ataque a distancia, sempre que tentar deve rodar 1d4 para ver se acerta seu alvo (4), se acerta algo a escolha do mestre (3, 2) ou erra (1).
  — — — — — // — — — — — — — — — // — — — — — — — — — // — — — — — —
- **Surdo**
  Um personagem surdo perde a capacidade de perceber aquilo fora do seu angulo de visão, recebendo -5 em percepção e ficando sempre aberto a ataques de oportunidade
  — — — — — // — — — — — — — — — // — — — — — — — — — // — — — — — —
- **Mudo**
  Um personagem mudo perde a capacidade de verbalizar, se tornando incapaz de usar qualquer pericia que necessite a fala
  — — — — — // — — — — — — — — — // — — — — — — — — — // — — — — — —
- **Alheio**
  Seus sentidos te abandonaram, você não percebe mais nada pode atacar a esmo mas o resultado sempre será utilizado em um alvo a escolha do mestre ou desperdiçado
  — — — — — // — — — — — — — — — // — — — — — — — — — // — — — — — —
- **Caído**
  Um inimigo caído sofre de algumas penalidades para se defender, perde 3 de defesa contra ataques físicos, mas ganha 3 de defesa contra ataques a distância, além de se mover apenas metade da sua velocidade de deslocamento e receber -5 de penalidade em reflexos
  — — — — — // — — — — — — — — — // — — — — — — — — — // — — — — — —
- **Envenenado**:
  Uma condição mais complexa e com muitos casos diferentes, porem caso a habilidade não especifique o tipo de veneno o efeito será sempre 1d4 de dano de veneno pela duração descrita na habilidade ou 1d6 rodadas
  — — — — — // — — — — — — — — — // — — — — — — — — — // — — — — — —
- **Doente**
  Outra condição a ser especificada pelas habilidades que a causam, normalmente dividida em estágios e sendo resolvida apenas por magia especifica para remoção de doenças ou remédios do doutor 
  — — — — — // — — — — — — — — — // — — — — — — — — — // — — — — — —
- **Sufocado** 
  Você não consegue respirar, perdendo temporariamente 1 de corpo por rodada, caso chegue a -3 sufoca até a morte
  — — — — — // — — — — — — — — — // — — — — — — — — — // — — — — — —
- **Vulnerável**
  Um alvo suscetível a ataques, recebendo -3 na ND e -2 em IP's possuidas, pode receber ataques de oportunidade em alguns casos
  — — — — — // — — — — — — — — — // — — — — — — — — — // — — — — — —
- **Desprevenido**
  Um alvo fácil, recebendo -5 na ND e -3 nas IP's possuidas, está suscetível a ataques de oportunidade e não pode utilizar a reação de esquiva ou aparar além de receber desvantagem em salvaguardas
  — — — — — // — — — — — — — — — // — — — — — — — — — // — — — — — —
- **Indefeso**
  incapaz de se defender de qualquer forma, desse modo não pode utilizar reações de defesa, recebe -10 na ND e - 7 nas IP's (não pode ser menor que 0) e é incapaz de realizar salvaguardas contra magias e habilidades
  — — — — — // — — — — — — — — — // — — — — — — — — — // — — — — — —
